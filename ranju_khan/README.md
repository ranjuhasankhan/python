# Ranju Khan Website

A simple Flask-based website for Ranju Khan with Home, About, and Contact pages. The contact form saves messages to a local SQLite database.

## Features
- Home, About, and Contact pages
- Contact form with validation
- Stores messages in SQLite database
- Responsive and clean design

## Setup
1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
2. Run the app:
   ```
   python app.py
   ```
3. Visit [http://127.0.0.1:5000](http://127.0.0.1:5000)

## Project Structure
- `app.py` - Main Flask application
- `templates/` - HTML templates
- `static/css/` - CSS styles
- `requirements.txt` - Python dependencies
